About This Extension

Colophon

This extension was developed using Google Chrome's™ Extension API in JavaScript and HTML 5.  Contributions to this project have been made by the public via GitHub as well as the following people and groups:

Extension Creator
  • Ryan Brodkin at Brodkin CyberArts
Technology and Design
  • Originally based on Mail Checker Plus by Anders Sahlin.
  • Forked from Mail Checker Minus by Andrew LeCody.
  • Panel icons supplied by Icon Shock.
Libraries
  • We give our life blood to jQuery for all its awesomeness.
  • Dates processed using Internet Timestamp Parser by Sebastiaan Deckers.
  • Form fields beautified by Uniform.
  • Templating enhanced with Simple JavaScript Templating by John Resig.

Change Log

1.3.3 (2011-12-11)

  • Changes in the session will now be autodetected.
  • Adjusting description to include newly added features.
  • Fixed issue in which plugin would fail to load on initial install.
  • Removed check to see if regular Gmail checking is disabled.
  • Adjusted manifest to use newer keys.
  • New audio notifications added.

1.3.2 (2011-12-10)

  • Fixed version change notification popup and storage of version number in database.

1.3.1 (2011-12-10)

  • Published test extension to Chrome Web Store.
  • More optimizations to backend.
  • Fixed links to inbox in dropdown menu.

1.3 (2011-12-1)

  • Forked and rebranding as Notifier Plus for Google Mail.
  • Brand spanking new interface that matches Google's new aesthetic.
  • Switched back to Local Storage since IndexedDB was over-complicating things and discouraged community contributions.
  • Options pages now significantly cleaner than they used to be.
  • Options have been rewritten to be clearer and more conversational.
  • Switched from Google's Basic HTML API to the much neer AJAX api.
  • State of starred messages is now maintained when popup is opened and closed.
  • Added support for Importance in popup.
  • Switching icons to use Google's official sprites for consistency.
  • Upgraded to jQuery v1.7.
  • Utilizing jQuery's new .on() method.
  • Removing support for Google Apps URLs, no longer needed with multiple sign-in.
  • Taking full advantage of HTML5 and CSS3 bundled into Google Chrome.

1.2.4.9 (2011-11-15)

  • Removed dead links and donation buttons.
  • Updated various pages to indicate that this extenion is open-source, released under the GPLv3.
  • Modified the logo to a show minus instead of plus.

1.2.4.8 (2011-09-25)

  • Fixed multiple account discovery

1.2.4.7 (2011-08-23)

  • Mailto-links will now be parsed on HTTPS sites as well
  • Fixed a bug causing the action buttons to malfunction

1.2.4.6 (2011-06-14)

  • Fixed an issue with GAFYD account management on the options page (they can now be deleted properly again)

1.2.4.5 (2011-04-30)

  • Fixed an issue with browser action icon rendering and animation

1.2.4.4 (2011-04-26)

  • Added a loading animation on the popup page when the extension is waiting for an outgoing request
  • Client storage backend rewritten (bye LocalStorage, hello IndexedDB).
    You will have to re-enter any custom settings, sorry!
  • New icon set, OSX (thanks to Marijn van der Werf)

1.2.4.2 (2011-04-21)

  • Added experimental one-way label support

1.2.4.1 (2011-04-16)

  • Using actions in the preview window will show the next (or previous) unread mail instead of returning to compact mode
  • Update alerts regarding the extension will be displayed in a notification instead of in a new tab

1.2.4.0 (2011-04-14)

  • Added previous- and next-links in expanded preview window to easily navigate between unread mail
